## Programation Orienté Objet et Modelisation

Il s'agit d'un depot git pour garder les Travails Dirigés du module POO et Modelisation en C++ au sein de la Université de Montpellier.

#### Compile and run
J'utilise [GCC GNU Compiler](https://gcc.gnu.org) pour tester chaque code.

```bash
g++ -o bin.exe main.cpp
```

Ou, en cas d'un code avec plusieurs classes:
```bash
g++ -o bin.exe main.cpp myclass.cpp
```

Et pour executer:
```bash
./bin
```
